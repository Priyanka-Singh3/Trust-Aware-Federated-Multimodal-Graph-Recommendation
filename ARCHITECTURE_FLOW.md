### STAGE 3: GNN LAYER (BIPARTITE GRAPH CONVOLUTION)

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                    BIPARTITE GRAPH NEURAL NETWORK                                   │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│   LAYER 0: INPUT                                                                    │
│   ┌─────────────────────────────────────────────────────────────────────────────┐  │
│   │                                                                             │  │
│   │      USERS (U)                    ITEMS (I)                               │  │
│   │                                                                             │  │
│   │    U1  U2  U3  U4                I1  I2  I3  I4                           │  │
│   │    ○   ○   ○   ○                 ○   ○   ○   ○                              │  │
│   │     ╲  │  ╱  │                  │╲  │  ╱│                                 │  │
│   │      ╲ │ ╱   │                  │ ╲ │ ╱ │                                 │  │
│   │       ╲│╱    │                  │  ╲│╱  │                                 │  │
│   │        ×     │                  │   ×   │     Bipartite edges:            │  │
│   │       ╱│╲    │                  │  ╱│╲  │     U-I interactions            │  │
│   │      ╱ │ ╲   │                  │ ╱ │ ╲ │                                 │  │
│   │     ╱  │  ╲  │                  │╱  │  ╲│                                 │  │
│   │                                                                             │  │
│   │   h_u^0 = Encoder output           h_i^0 = Encoder output                  │  │
│   │                                                                             │  │
│   └─────────────────────────────────────────────────────────────────────────────┘  │
│                                    │                                                │
│                                    ▼                                                │
│   LAYER 1: MESSAGE PASSING                                                          │
│   ┌─────────────────────────────────────────────────────────────────────────────┐  │
│   │                                                                             │  │
│   │   Message: m_{u←i} = h_i^0 ⊗ W_0      (from items to users)                │  │
│   │   Message: m_{i←u} = h_u^0 ⊗ W_0      (from users to items)                │  │
│   │                                                                             │  │
│   │   Aggregation: h_u^1 = σ(Σ_{i∈N(u)} m_{u←i} / |N(u)|)                       │  │
│   │   Aggregation: h_i^1 = σ(Σ_{u∈N(i)} m_{i←u} / |N(i)|)                       │  │
│   │                                                                             │  │
│   │   Where: σ = ReLU activation, N(·) = neighbors                               │  │
│   │                                                                             │  │
│   └─────────────────────────────────────────────────────────────────────────────┘  │
│                                    │                                                │
│                                    ▼                                                │
│   LAYER 2: FINAL EMBEDDINGS                                                         │
│   ┌─────────────────────────────────────────────────────────────────────────────┐  │
│   │                                                                             │  │
│   │   h_u^2 = GNN_Layer_2(h_u^1, h_i^1)  ──┐                                   │  │
│   │   h_i^2 = GNN_Layer_2(h_i^1, h_u^1)  ──┼──▶ Final user/item embeddings    │  │
│   │                                         │                                   │  │
│   │   Final Score = dot(h_u^2, h_i^2)     │                                   │  │
│   │                                                                             │  │
│   └─────────────────────────────────────────────────────────────────────────────┘  │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

## 7. TRAINING WORKFLOW

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                         TRAINING PHASE FLOW                                        │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│   FOR EACH COMMUNICATION ROUND (t = 1 to 10):                                      │
│                                                                                     │
│   ┌─────────────┐                                                                   │
│   │  SERVER     │  1. Broadcast θ_global^(t) to all clients                        │
│   │             │  2. Wait for updates                                             │
│   └──────┬──────┘                                                                   │
│          │                                                                          │
│          ▼                                                                          │
│   ┌─────────────────────────────────────────────────────────────────────────┐      │
│   │ PARALLEL ON EACH CLIENT (k = 1 to 5):                                 │      │
│   │                                                                         │      │
│   │   a) Receive θ_global^(t)                                             │      │
│   │   b) Load local data batch                                              │      │
│   │   c) FOR e = 1 to 5 (local epochs):                                     │      │
│   │        i.   Forward pass (Encoder → GNN → Trust → Score)              │      │
│   │        ii.  Compute loss: L = MSE(pred, true_rating)                     │      │
│   │        iii. Backward pass: ∇L                                          │      │
│   │        iv.  Optimizer step: θ ← θ - η·∇L                                │      │
│   │   d) Add DP noise: θ_k^(t)' = θ_k^(t) + N(0, σ²)                         │      │
│   │   e) Send θ_k^(t)' to server                                             │      │
│   │                                                                         │      │
│   └─────────────────────────────────────────────────────────────────────────┘      │
│          │                                                                          │
│          ▼                                                                          │
│   ┌─────────────┐                                                                   │
│   │  SERVER     │  3. Aggregate: θ_global^(t+1) = Σ (n_k/N) · θ_k^(t)'             │
│   │             │  4. Validate on hold-out set                                     │
│   │             │  5. IF converged: STOP, ELSE: t ← t+1, repeat                    │
│   └─────────────┘                                                                   │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```
